<h2>Tags semânticas</h2>
Representam o conteúdo da página. Possuem a mesma função de uma div porém facilita a interpretação.

```html
<header></header>
<section></section>
<nav></nav>
<main></main>
<footer></footer>
<summary></summary>
<article></article>
etc...
```
